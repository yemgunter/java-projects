PROGRAM SPECIFICATIONS

Beginning with the files that you have downloaded named Proj03.java and Proj03Skeleton.txt, create a new file named Proj03Runner.java to meet the specifications given below.

Note that you must not modify the code in the file named Proj03.java.

You may rename and modify the code in the file named Proj03Skeleton.txt to produce the required output if you choose to do so. If you take that approach, rename the file Proj03Runner.java and see the additional instructions below.

Be sure to display your name in the output where indicated.

When you place both files in the same folder, compile them both, and run the file named Proj03.java, the program must display the text shown below on the command line screen.

The numeric output values are based on a random number generator and will differ from one run to the next. However, in all cases, the first two numbers must match and must be one less than the third number.

128
I certify that this program is my own work
and is not the work of others. I agree not
to share my solution with others.
Replace this line with your name

128
129

===== Additional Instructions=====

The program structure required by this assignment consists of two class files that work together to produce the required output. The file named Proj03Skeleton.txt is provided to help you create that structure. You may find the following steps useful in helping you to understand the structure and to write a correct solution for the assignment.

1. Copy the files named Proj03.java and Proj03Skeleton.txt into an empty folder.

2. Change the name of Proj03Skeleton.txt to Proj03Runner.java

3. Study the code in the file named Proj03 very carefully. Make certain that you understand and can explain the behavior of the last three statements  in that code. (You don't need to understand the code that generates the random number at this point as long as you understand the comments.) If there is something that you don't understand in that code, please let me know.

4. Study the code in the file named Proj03Runner very carefully. Make certain that you understand and can explain the behavior of every statement in that code. If there is something that you don't understand in that code, please let me know.

5. Think about how the code in the file named Proj03Runner fits or meshes with (or should fit and mesh with) the code in the file named Proj03, taking into account that the code in the file named Proj03 will be executed first. The code in the file named Proj03Runner will be executed under control of the code in the file named Proj03.

6. Try to compile the file named Proj03.java. You will get the following error indicating that the two sets of code do not fit or mesh perfectly. (Note that the line number may differ from that shown below.)

Proj03Runner.java:14: error: incompatible types: 
possible lossy conversion from int to byte 
byte myVar = valIn; 
1 error

7. Modify the code in the file named Proj03Runner to eliminate that error when you compile the file named Proj03.java. When you are successful in eliminating that error, you should have created the files named Proj03.class and Proj03Runner.class in that same folder with no compiler errors.

8. Read the above specifications VERY CAREFULLY and compare the required output with the actual output.

9. Make any other modifications that may be needed to the file named Proj03Runner.java to cause your output to meet those specifications. Pay particular attention to the required numeric values.

10. Study the structure and the relationship between the two files and make certain that you understand it. Most of your assignments in this course will be based on this general structure.
