PROGRAM SPECIFICATIONS

Beginning with the files that you have downloaded named Proj04.java and Proj04Skeleton.txt, create a new file named Proj04Runner.java to meet the specifications given below.

Note that you must not modify code in the file named Proj04.java.

You may rename and modify the code in the file named Proj04Skeleton.txt to produce the required output if you choose to do so. If you take that approach, rename the file Proj04Runner.java.

Be sure to display your name in the output where indicated.

When you place both files in the same folder, compile them both, and run the file named Proj04.java, the program must display the text shown below on the command line screen.

The number of elements in the array is based on a command-line argument and may differ from one run to the next. The other numeric output values are based on a command-line argument/random number generator and may differ from one run to the next.

ADDITIONAL CLARIFICATION

A student in an earlier semester made me aware that he or she was confused as to how to proceed on this assignment. The advice that I gave is shown below.

For this assignment you need to do the following:

  1.  Read and understand the comments in the file named Proj04.java.
  2.  Copy the contents of the file named Proj04Skeleton.txt into a new file named Proj04Runner.java.
  3.  Open the file named Proj04Runner.java in your text editor.
  4.  Do not modify the line that reads static double run(byte[] byteArray){
  5.  Insert the new code necessary to satisfy the requirements explained in the comments.
  6.  Attempt to compile the file named Proj04Runner.java
  7.  Correct compiler errors if any.
  8.  Copy the file named Proj04.java into the same folder as the file named Proj04Runner.java
  9.  Attempt to compile the file named Proj04.java
  10.  If there are compiler errors, the errors will be in the file named Proj04Runner.java. Correct them.
  11.  Run the file named Proj04.java. Use 3 2 for the command-line arguments. That will make it easy for you to confirm that your arithmetic is correct. (You could use any value for the first argument but the second argument needs to be 2 in order to create a two-element array.)
  12.  Confirm that the program produces the correct output. If not, correct the code in the file named Proj04Runner.java. Compile and run again.
  13.  Repeat this process until your program produces the correct output.
  14.  Encapsulate your file named Proj04Runner.java in a zip file and submit through Blackboard.
