The purpose of this assignment is to assess the student's ability to write a simple Java program involving command-line arguments.

PROGRAM SPECIFICATIONS

Create a source code file named Proj02.java. Use that file to write a program named Proj02. To begin, your program must produce the output text shown below on the command-line screen. 

I certify that this program is my own work
and is not the work of others. I agree not
to share my solution with others.
Replace this line with your name

After that, your program must access, display, compute, and display the sum of an arbitrary number of integer command-line arguments. For example, if I were to use your program named Proj02.java and execute the following commands:

javac Proj02.java
java Proj02 1 2 3 4

your program would produce the following output on the command-line screen:

I certify that this program is my own work
and is not the work of others. I agree not
to share my solution with others.
Replace this line with your name

1
2
3
4
10

where 10 is the sum of the four integer command-line arguments.

Encapsulate the file named Proj02.java in a zip file and submit it via Blackboard. Note that your class must also be named Proj02.
